## üìù Overview

Avaya Proactive Outreach Manager (POM) is an outbound automation system used to make auto-dial calls, send SMS, and email to customers.  
In this project, I built a full automation testing framework to:

- Test the POM Manager web interface
- Automate the creation and execution of outbound campaigns
- Test Contact Strategy flows and retry rules
- Validate dashboard & real-time campaign monitoring
- Verify dialer modes, call results, and integration with Avaya AEP

This automation suite reduced manual testing effort by **70%** and improved stability for large-scale outbound campaigns.

## üéØ Automation Scope

### **1. Login & Permission Testing**

- Validate login using Admin, Supervisor, and Campaign Manager roles
- Permission-based UI and feature access

### **2. Outbound Campaign Automation**

- Create outbound campaigns automatically
- Upload customer phone list
- Set dialer modes (Predictive, Preview, Progressive)
- Validate retry logic, call results, and status transitions
- Start / Pause / Stop campaigns through automation

### **3. Contact Strategy Testing**

- Validate workflow for retry attempts
- Validate the waiting interval between retries
- Multi-channel flow testing (Call ‚Üí SMS ‚Üí Email)

### **4. Dashboard & Monitoring**

- Validate call counts and progress
- Monitor agent state
- Validate KPI metrics in real-time

### **5. Reporting Module**

- Download and validate POM reports
- Validate call disposition accuracy
- Match campaign results with report data
